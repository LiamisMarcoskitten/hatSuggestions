HatVision — AI Hat Style Detector

About the Project

HatVision uses a Teachable Machine image model to analyze your webcam feed in real time. It detects the type of hat you are wearing and provides a clear description of that style.

This project demonstrates real-time computer vision, AI classification, and interactive user feedback using TensorFlow.js.
